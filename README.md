# ResolveIT
### Smart Public Transport Grievance and Feedback Management System

## Abstract
ResolveIT is a web-based platform designed to improve grievance handling in public transportation systems. The system allows passengers to submit complaints, track complaint status, and provide feedback through a secure and user-friendly interface. It enhances transparency, reduces manual processes, and improves communication between passengers and transport authorities.

## Problem Statement
In traditional public transport systems, passengers often face difficulties in registering complaints and tracking their resolution. The process is usually manual, fragmented, and lacks transparency. This results in delayed responses and reduced passenger satisfaction.


## Features
- Passenger complaint registration
- Complaint status tracking
- Feedback and rating system
- Secure user authentication
- Admin dashboard for complaint management
